# Job Application Board

A Spring Boot web application for posting and viewing job listings.

## Prerequisites

- Java 17 or higher
- Maven 3.6+

## Running the Application

1. Clone the repository and navigate to the project folder

2. Run the application:
   - On Linux/Mac: `./mvnw spring-boot:run`
   - On Windows: `mvnw.cmd spring-boot:run`

3. Open your browser and navigate to `http://localhost:8080`

## Features

- **Home Page** - Navigate to view all jobs or add a new job
- **Add Job** - Post a new job with profile, description, experience requirements, and tech stack
- **View All Jobs** - Browse all posted job listings
